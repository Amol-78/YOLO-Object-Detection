# YOLO Object Detection Web Application

A modern web application for real-time object detection using YOLO models with a beautiful UI and Docker deployment.

## Features

-  **Image Detection**: Upload and detect objects in images
-  **Video Processing**: Process videos with object detection
-  **Webcam Support**: Real-time detection from webcam
-  **Multiple Models**: Support for YOLOv3-Tiny and YOLOv8n
-  **Customizable Settings**: Adjust confidence, NMS thresholds, colors
-  **Modern UI**: Clean, responsive interface with dialog boxes
-  **Audio Alerts**: Sound notifications on detection
-  **Responsive Design**: Works on desktop and mobile
-  **Docker Deployment**: Easy deployment with Docker
